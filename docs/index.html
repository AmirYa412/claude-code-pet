🐾 claude-pet

A pixel-art Claude that lives in your statusline and tells you — through sheer existential dread — when it's time to /compact.

What is it?

A single self-contained bash script that prints one 5-cell mascot frame each time Claude Code refreshes your statusline (~once a second). It blinks, glances, and emotes — a pixel-art Claude with a chunky orange body — and reacts to how full your context window is.

Why it's nice

  • Zero forks, zero temp files. Pure bash builtins + $RANDOM.
  • Never loops. Weighted-random frames feel alive, not metronomic.
  • Context-aware. Three tiers — relaxed (0–40%), exhausted (41–69%), panic (70%+).
  • No jitter. Every frame is exactly 5 terminal cells.
  • One file. Drop it in, point your statusline at it.
  • Mostly idle. Rests, blinks, glances far more than it emotes.

Install

One command — needs Claude Code and jq (brew install jq / sudo apt install jq). On Windows, run it in Git Bash or WSL — not PowerShell.

curl -fsSL https://raw.githubusercontent.com/AmirYa412/claude-code-pet/main/claude-pet-install.sh | bash

Cloned the repo instead? Run bash claude-pet-install.sh. It drops the pet + wrapper into ~/.claude and wires settings.json (with a backup).
